Pre-Deployment (24 hours before)\n```bash\n# Build for production\nnpm run build\n\n# Test build locally\nnpm run preview\n\n# Run security audit\nnpm audit\n\n# Check for unused code\nnpm run analyze\n```\n\n### 2. Environment Setup\n```bash\n# Copy environment template\ncp .env.production.template .env.production\n\n# Edit with production values\nvim .env.production\n\n# Verify key environment variables\necho $VITE_API_URL\necho $VITE_IS_PRODUCTION\n```\n\n### 3. Database Migration\n```bash\n# Backup current database\npg_dump production_db > backup_$(date +%Y%m%d).sql\n\n# Run migrations\nknex migrate:latest --env production\n\n# Verify database\npsql production_db -c \"SELECT * FROM users LIMIT 1;\"\n```\n\n### 4. Deployment\n```bash\n# Deploy using CI/CD pipeline\ngit push production main\n\n# Or manual deployment\nscp -r build/* user@production.server:/var/www/chime/\nssh user@production.server \"systemctl restart chime\"\n```\n\n### 5. Post-Deployment (First 24 hours)\n- [ ] Monitor error rates (should be < 0.1%)\n- [ ] Check performance metrics (response time < 200ms)\n- [ ] Verify all features working\n- [ ] Test user flows:\n  -
